## [Unreleased]

### Added

- Monorepo scaffold: pnpm workspaces + Turborepo, OXC toolchain (oxlint + oxfmt), strict TypeScript, Vitest with a 90% coverage gate, CI.
- `@lagda/core` — domain types and Zod-validated config (PostgreSQL-only).
- `@lagda/auth-contract` — token/claims types and the owner/admin/user RBAC permission matrix.
- `@lagda/ui` — minimalist, neutral shadcn design system (Tailwind preset + components + Storybook).
- `@lagda/db` — Kysely Postgres data layer, typed schema, and migrations.
- `@lagda/templating` — MJML rendering with safe variable interpolation.
- `@lagda/connectors` — `ConnectorInterface` with a Google Workspace implementation and a Microsoft stub.
- `@lagda/jobs` — pg-boss queue with `syncDirectory`, `deploySignature`, and `notify` jobs.
- `@lagda/logger` — pino structured logging with PII redaction.
- `apps/server` — Hono application monolith serving the SPA and the typed API, with OpenAPI docs via Scalar.
- `apps/web` — Vite + React SPA consuming the typed Hono RPC client.
